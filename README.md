## Overview

pharmacy-mgmt-system is an all-in-one pharmacy management platform that integrates backend APIs with a dynamic frontend to streamline healthcare operations. Designed for developers, it offers a scalable architecture to handle inventory, prescriptions, supply chain, and user management seamlessly.

### Why pharmacy-mgmt-system?

This project empowers developers to build robust healthcare applications with ease. The core features include:

- :pill: RESTful API Endpoints: Manage medicines, prescriptions, suppliers, and sales efficiently.
- :zap: Real-Time Data Handling: Keep inventory, sales, and supply chain information up-to-date.
- :jigsaw: Modular Architecture: Easily extend and customize to fit complex healthcare workflows.
- :lock: Secure Role-Based Access: Authenticate doctors, sellers, and admins for protected operations.
- :dart: User-Friendly UI: Seamless navigation and dynamic updates for an optimal user experience.

This project requires the following dependencies:

- **Package Manager:** Npm

### Installation

Build pharmacy-mgmt-system from the source and install dependencies:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Pakorn/pharmacy-mgmt-system

2.	**Navigate to the project directory:**
    ```bash
    cd pharmacy-mgmt-system

3.	**Install the dependencies:**
Using npm:
    ```bash
    npm install
